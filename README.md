# Cake.Tfx

[![License](http://img.shields.io/:license-mit-blue.svg)](http://cake-contrib.mit-license.org)

Cake.Tfx is an Addin for [Cake](http://cakebuild.net/) which allows the execution of the tfx command line tool.
